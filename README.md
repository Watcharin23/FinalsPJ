# ระบบผู้ช่วยแบบตัวแทนเชิงปัญญาประดิษฐ์สำหรับองค์กร
## AI Agent Assistant System for Organizations

> ปริญญานิพนธ์ ภาคการศึกษา 2568

---

## 📋 เกี่ยวกับโปรเจกต์

ระบบผู้ช่วยแบบตัวแทนเชิงปัญญาประดิษฐ์ (AI Agent) ที่พัฒนาขึ้นเพื่อช่วยเหลือและสนับสนุนการทำงานภายในองค์กร โดยใช้เทคโนโลยี Artificial Intelligence และ Natural Language Processing ในการตอบคำถาม ประมวลผลข้อมูล และช่วยเหลือในงานต่างๆ

### วัตถุประสงค์

- พัฒนาระบบ AI Agent ที่สามารถเข้าใจและตอบสนองต่อความต้องการของผู้ใช้ในองค์กร
- เพิ่มประสิทธิภาพการทำงานและลดเวลาในการค้นหาข้อมูล
- สร้างระบบที่สามารถเรียนรู้และปรับตัวตามบริบทขององค์กร
